# Solar Buy-Side — Landing Page

Landing page do infoproduto **Manual Solar Buy-Side** (HTML/CSS/JS puro, sem build).

## Rodar localmente

Recomendado usar um servidor local (para garantir carregamento correto de assets):

```powershell
python -m http.server 5173
```

Depois acesse `http://localhost:5173`.

## Arquivos principais

- `index.html`
- `assets/css/styles.css`
- `assets/js/main.js`
- `docs/design-system.md`

## Personalização rápida

- Checkout: altere o link do botão de compra em `index.html` (seção `#oferta`).
- WhatsApp: altere o link `https://wa.me/...` em `index.html` (seção Contato).
- Depoimentos e fotos: substitua os SVGs em `assets/img/` por imagens reais (WebP recomendado).
